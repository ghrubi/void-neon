# Hardware

## Primary Development Machine

MacBook Air 2017

Running:

- Void Linux
- Hyprland
- Ghostty

---

## Trackpad

Configuration:

- tap-to-click disabled
- tap-and-drag disabled
- disable while typing enabled

The desktop assumes physical clicks only.

---

## Brightness

Custom shell scripts.

Supports:

- Screen brightness
- Keyboard backlight

Permissions managed through udev rules.

---

## Volume

Uses:

wpctl

Custom OSD.

---

## Power Profiles

Uses:

power-profiles-daemon

Waybar displays:

- current profile
- tooltip
- click to cycle

Future:

Automatically switch to Power Saver while on battery.

---

## Goal

Void Neon should feel designed specifically for Intel MacBooks while remaining portable to other hardware.
