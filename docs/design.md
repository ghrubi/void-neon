# Void Neon Design

## Philosophy

Void Neon is intentionally minimal.

Not minimal in features.

Minimal in visual noise.

---

## Colors

Background:

```
#000000
```

Accent:

```
#00e5ff
```

Surfaces:

```
Surface
Elevated
Control
```

Layered using progressively lighter blacks.

---

## Typography

Inter is used everywhere possible.

Small fonts are preferred.

Dense information density is encouraged.

---

## Visual Language

Everything should look related.

Examples:

- Ghostty
- Waybar
- Mako
- Fuzzel
- Neovim

should all appear to belong to the same desktop.

---

## OSD

Hardware notifications are intentionally simple.

Examples:

```
☀ ███████ 70%

🔊 ██████ 60%

⌨ ███ 30%
```

Single-line.

No unnecessary titles.

---

## Launcher

Fuzzel provides:

- applications
- clipboard
- files

Fast.

Minimal.

Keyboard-first.

---

## Windows

Floating windows are used for temporary tools.

Examples:

- Yazi
- LazyGit
- Bluetui

Tiling is reserved for primary work.
