# Void Neon

A reproducible, keyboard-first desktop environment for Void Linux on Intel MacBooks.

## Design

- Pure black surfaces
- Cyan accents
- Inter typography
- Hyprland
- Ghostty
- Neovim
- Yazi
- Fuzzel

## Status

Early development.

## Goals

- Reproducible installs
- Minimal dependencies
- Machine-specific polish
